# Incoming Kelel Assets

Drop real Kelel website assets into this folder for the next authenticity pass.

Recommended first files:

- `kelel-leadership-hero.jpg`
- `kelel-team-review.jpg`
- `kelel-delivery-team.jpg`
- `kelel-project-screenshot-1.png`
- `kelel-project-screenshot-2.png`
- `kelel-certificate-1.pdf`
- `kelel-registration.pdf`
- project screenshots as `.png` or `.jpg`
- certificates or registration files as `.pdf`, `.png`, or `.jpg`

Suggested structure:

- `incoming-assets/leadership/`
- `incoming-assets/team/`
- `incoming-assets/projects/`
- `incoming-assets/certificates/`

Important guidance:

- Use only files approved for public website use
- Prefer landscape images where possible
- Prefer width of at least 1600px for hero and section visuals
- Keep original filenames descriptive
- If a file is sensitive or internal-only, do not place it here
- Fill in `asset-manifest-template.csv` when handing over real files so each asset has usage notes and placement guidance
